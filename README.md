TradeMap — Автоматизированная система геомаркетингового анализа торговых площадок.

Данный проект реализует программ разработанную в рамках магистерской диссертации "автоматизированная информационная система для выбора портфеля торговых площадок на основе геомаркетинга" (ВолгГТУ, 2026).

Система позволяет:
- Собирать и обрабатывать открытые геопространственные данные (OpenStreetMap).
- Строить полигоны Тиссена с учётом пространственных барьеров.
- Выявлять территории с недостаточной торговой инфраструктурой.
- Учитывать сетевую доступность и реальную структуру городской среды, которая важна для линейных городов.
- Визуализировать результаты анализа в удобном веб-интерфейсе.

Основные возможности программы:
- Интерактивная карта торговых объектов с полигонами Тиссена.
- Алгоритм формирования портфеля торговых площадок.
- Учёт пространственных барьеров (реки, железные дороги, овраги и т.д.).
- Аналитика территорий с недостаточной торговой инфраструктурой.
